# React Product List Filters
A dynamic and responsive React application designed to display a list of products with advanced,
real-time filtering and sorting capabilities.
This project serves as a clean baseline or reusable component system for e-commerce interfaces,
demonstrating efficient state management and performance-optimized rendering in React.

## Getting Started

 **Clone the repository:**
   ```bash
   git clone [https://github.com/Katsikas/react-product-list-filters.git](https://github.com/Katsikas/react-product-list-filters.git)
```

## Backend

In your root folder run the following commands:
```
python -m venv venv
venv\Scripts\activate
```
This will activate the venv. 
Then while on the venv run the following:
```
pip install -r requirements.txt
```
When installation is over the following command will serve the backend:
```
cd backend
python manage.py runserver
```

## Frontend:
```
cd frontend
npm install
npm run dev
```
